# 变更日志

遵循 Keep a Changelog 格式，并使用语义化版本。

## [Unreleased]

### Added - 2026-07-18

- 新增“微醺小记”顶栏入口、独立内容集合、酒评列表和详情路由。
- 新增多图相册、五星评分、品饮信息、酒类标签筛选、空状态和草稿过滤。
- 新增 Pages CMS 配置，可在浏览器中管理博客、动态与酒评，并将图片和 Markdown 提交回 GitHub。
- 新增 `docs/CONTENT_EDITOR.md`，记录仅仓库所有者可用的写作与发布流程。

### Changed - 2026-07-18

- 延续现有液态玻璃、橙色强调和深浅主题，为微醺页面补充移动端宽度约束。
- Pages CMS 保存内容时启用字段合并，避免覆盖现有但未显示在编辑表单中的 Frontmatter 字段。

### Added

- 在 `public/projects/llmtrans/` 接入完整的 LLMtrans 产品展示子站。
- 保留四个产品页面、独立背景视频、团队头像、客户端邀请码验证和 Release 下载入口。
- 增强 LLMtrans 子站在移动端的背景视频兼容，隐藏原生 WebKit 控件并在首次触摸或页面恢复时重试播放。

### Changed

- 将博客顶栏的“在线游戏”入口改为“参与项目”，并链接到 `/projects/llmtrans/`。
- 在“参与项目”旁恢复独立的“在线游戏”入口，继续提供 `/games` 下的合成大南开游戏。
